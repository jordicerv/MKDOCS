    Implementació i configuració d’un entorn de domini amb Active Directory

    En aquest apartat veurem detalladament què és Active Directory (AD) i com pot ajudar a gestionar de forma centralitzada els recursos tecnològics dins d’una organització.
    AD és un sistema de directori creat per Microsoft que permet controlar comptes d’usuari, ordinadors, permisos i altres elements dins d’una xarxa empresarial.

    Desplegament del domini Active Directory

    Com a pas inicial, cal verificar que el servidor virtual disposa de dues interfícies de xarxa: una configurada com NAT i l’altra com a Xarxa Interna.

    Amb la màquina virtual en marxa, accedim a les opcions de xarxa i comprovem que cap IP estigui assignada automàticament. Procedim a configurar una IP estàtica de forma manual.

    Ens situem a les propietats de l’adaptador de xarxa, obrim la configuració IPv4 i hi introduïm manualment la IP, la passarel·la per defecte i els servidors DNS.

    Verifiquem amb CMD que la configuració manual s’ha aplicat correctament.

    Configuració inicial del domini

    Per començar, obrim l’Administrador del servidor.

    Anem a l’opció Administrar i escollim Afegir rols i característiques.

    L’assistent de configuració s’iniciarà per guiar-nos en el procés d’instal·lació.

    Aquí veurem una llista amb els servidors disponibles. Triem aquell on volem desplegar el servei de domini.

    En aquest pas, és clau seleccionar les funcionalitats següents:

    • Active Directory Domain Services
    • DNS Server
    • (Opcional) DHCP Server si volem assignació automàtica d’IPs

    També cal assegurar-se que la funcionalitat Gestió de directrius de grup (GPO) estigui activada. En general, ja ho està per defecte.

    Pots continuar avançant per les pantalles de l’assistent fins arribar a la confirmació. Revisa els paràmetres i clica Instal·lar.

    Quan el procés d’instal·lació finalitzi, prem Promoure aquest servidor a controlador de domini.

    Creació d’un nou bosc de domini (forest)

    Un bosc és l’estructura superior d’Active Directory i pot contenir múltiples dominis.
    Seleccionem Afegir un bosc nou i introduïm el nom del domini (per exemple: institut.local).

    El sistema ens demanarà una contrasenya per al compte d’administrador del domini. Guarda-la bé.

    Durant la configuració del DNS, pot aparèixer un avís. Es tracta d’un missatge habitual. Clica Següent.

    Cal establir el nom NetBIOS del domini. Pot ser un nom curt que representi la nostra xarxa (ex: INSTITUT).

    A la darrera pantalla, veurem un resum amb un script de configuració. Es pot guardar per reutilitzar-lo en el futur.

    Ara s’inicien les comprovacions finals. Un cop tot estigui correcte, prem Instal·lar. En acabar, el sistema es reiniciarà automàticament.

    Quan l’equip es torni a iniciar, accedim amb l’usuari i contrasenya establerts. Normalment, l’usuari és Administrador.
