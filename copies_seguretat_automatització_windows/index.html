    Còpies de seguretat i automatització a Windows

    Introducció

    En aquest apartat aprendrem a fer còpies de seguretat a Windows i a automatitzar el procés mitjançant scripts i polítiques del sistema.

    Preparació de l’entorn

    El primer pas és afegir un disc nou a la màquina virtual. Un cop afegit, el formategem en NTFS i hi creem un directori per emmagatzemar les còpies.

    Exemple: dins del nou volum, creem una carpeta per desar-hi les còpies d’usuari.

    Script per fer la còpia del perfil d’usuari

    A continuació, creem un script .bat per copiar automàticament el perfil de l’usuari actiu cap a una ubicació al disc nou. Exemple amb unitat G:

    @echo off
mkdir "G:\CopiesUsuaris\%USERNAME%"
xcopy "C:\Users\%USERNAME%" "G:\CopiesUsuaris\%USERNAME%" /E /H /C /Y

    Significat dels paràmetres utilitzats en XCOPY

    • /E – Inclou totes les subcarpetes, fins i tot les que estan buides.
    • /H – Copia fitxers ocults i de sistema.
    • /C – No s’atura si es troba algun error.
    • /Y – Evita preguntes de confirmació quan hi ha sobreescriptura.

    Exemple de resultat esperat: Si l’usuari és alumne1, es crearà G:\CopiesUsuaris\alumne1\ amb tot el seu perfil copiat.

    Automatitzar l’script amb les polítiques de Windows

    Per automatitzar l’execució del script a l’inici de sessió, utilitzem l’Editor de polítiques de grup local (gpedit.msc):

    Ruta:
    Configuració de l'usuari → Configuració de Windows → Scripts (Inici / Tancament de sessió) → Inici de sessió

    1. Obre “Inici de sessió” amb doble clic.
    2. Fes clic a Afegir...
    3. Selecciona el teu fitxer .bat a través de Navega...

    Aplicar les polítiques immediatament

    Executa la següent comanda en una consola amb privilegis d’administrador:

    gpupdate /force

    Com fer-ho:

    • Prem Windows + R, escriu cmd

    • Prem Ctrl + Shift + Enter per obrir com a administrador

    Què fa aquesta comanda?

    • Actualitza totes les polítiques del sistema.

    • Amb l’opció /force, re-aplica totes les polítiques, encara que no s’hagin modificat.

    Verificació del funcionament

    Quan iniciïs sessió com l’usuari alumne1, el sistema executarà el script i es crearà automàticament la còpia del seu perfil a la carpeta corresponent.
